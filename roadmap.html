Roadmap

What happens next, probably

The project is still changing quickly. This roadmap is a useful compass, not a carved tablet.

Stages

Stage 1 - Public source / developer alpha

Status: mostly active.

Source repos, package generation, review sources, and curriculum specs are being built in public.

Stage 2 - Public online preview

Status: online and under construction.

Accounts, curriculum reading, package installation, and review access are online. The web version remains rough, unfinished software, with no fake launch party.

Stage 3 - Public beta

Later.

Useful enough for brave learners, but still honest about rough edges.

Stage 4 - Packaged releases

Later.

Installable releases once the CLI, content packages, and review flow stop changing every five minutes.

Near-term Roadmap

  • Polish Dutch Reading formatting across the available Chapters 1-25.
  • Review the possible Chapter 18 issue once it is specified.
  • Apply the resulting Reading format to Vietnamese.
  • Establish that format as the canonical Reading structure.
  • Add reader controls for showing or hiding translations.
  • Continue deriving Decks from approved Reading content.
  • Add derived Grammar and Exercises later.
  • Keep periodic human editorial review in the content-production workflow.

Packaging Roadmap

  1. Docker image - available as an experimental public alpha
  2. Gentoo package validation
  3. Arch/AUR
  4. Debian/Ubuntu .deb
  5. Fedora/openSUSE/RHEL-family .rpm
  6. Generic Linux tarball/AppImage
  7. Maybe Flatpak/Snap/desktop wrapper later
