Roadmap

What happens next, probably

The project is still changing quickly. This roadmap is a useful compass, not a carved tablet.

Stages

Stage 1 - Public source / developer alpha

Status: mostly active.

Source repos, package generation, review sources, and curriculum specs are being built in public.

Stage 2 - Public online preview

Goal: read-only website/frontend preview.

A lightweight site and later a read-only content preview. No account system. No fake launch party.

Stage 3 - Public beta

Later.

Useful enough for brave learners, but still honest about rough edges.

Stage 4 - Packaged releases

Later.

Installable releases once the CLI, content packages, and review flow stop changing every five minutes.
