2026-07-11

WhackSmacker Public Alpha Is Online

Human Voice

A lot got done, but the most important thing is that it's all online now. The rest is just fiddling in the margins. If you want to use this, use the CLI for now, that one is actually pretty usable. The web frontend is just pre-alpha.

The alpha is genuinely online

WhackSmacker 0.1.0-alpha.1 is now publicly deployed at alpha.whacksmacker.com. The existing website at www.whacksmacker.com remains unchanged; that complete website move, /developer-notes/ integration, and the final public experience are still later work.

The practical recommendation is still the CLI. It is the usable interface today. The web frontend is pre-alpha: useful for seeing where the project is going, but not a polished or generally usable interface yet. The logged-in source-language control is also known follow-up work.

A real multi-user deployment

The deployed application runs in PostgreSQL multi-user mode, with no public registration. Password authentication, persistent sessions, CSRF protection, Origin checking, login throttling, and per-user state isolation are active. The PostgreSQL migrations completed successfully, and both the production application and database containers are healthy.

Public traffic reaches the deployment through Nginx on the DigitalOcean VPS, while Nginx reaches the WhackSmacker backend over the public-services ZeroTier network. HTTPS is active for the alpha domain.

Image, packages, and curriculum

The Docker image was built, validated, and published as sleepiestmario/whacksmacker:0.1.0-alpha.1 and sleepiestmario/whacksmacker:alpha. The package feed was rebuilt successfully with the current language packages, and the curriculum system now enforces the canonical thirty-person recurring casts.

Checks before going online

The complete ordinary test suite passed with 359 passing tests, zero failures, and one deliberately skipped PostgreSQL integration test. PostgreSQL behavior had also been tested separately. That does not make the frontend finished, but it does mean this milestone is more than a mock-up: the system is deployed, healthy, and available.

See the WhackSmacker source on GitHub or open the public alpha.
